CodeAlpha_Chatbot_for_FAQs

An intelligent, console-based Chatbot application designed to resolve frequently asked questions regarding the CodeAlpha internship platform. This project utilizes Natural Language Processing techniques to accurately parse intent and fetch contexts, fulfilling Task 2 of the CodeAlpha Artificial Intelligence Internship program.

Features
- Intent Matching: Utilizes Term Frequency-Inverse Document Frequency vector space modeling to digitize language parameters.
- Mathematical Verification: Employs Cosine Similarity dot-product analysis to mathematically gauge the closest matching context.
- Fallback Safeguards: Implements constraint thresholds to cleanly filter out non-matching queries.
- Instant Local Runtime: Runs lightweight executions directly in terminal consoles.

Tech Stack & Dependencies
- Core Engine: Python 3.x
- Mathematical Modeling: scikit-learn

Local Installation & Setup

1. Navigate to the Directory:
   cd "Task 2_Chatbot_for_FAQs"

2. Install Required Extensions:
   pip install -r requirements.txt

3. Run Application Runtime:
   python app.py
