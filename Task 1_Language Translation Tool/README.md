CodeAlpha_Language_Translation_Tool

A desktop-based Language Translation application built using Python, Tkinter, and the Deep Translator framework. This project fulfills Task 1 of the CodeAlpha Artificial Intelligence Internship program.

Features
- Auto-Detect Language: Automatically parses input configurations without requiring explicit source assignment.
- Multiple Target Formats: Supports high-fidelity translation rendering across multiple modern global languages.
- Clipboard Accessibility: Features an integrated functional copy utility module allowing immediate utilization of translated contents.
- User-Friendly Interface: Built with structured layouts matching desktop tool usability standards.

Tech Stack & Dependencies
- Core Engine: Python 3.x
- GUI Engine: Tkinter
- API Wrapper Engine: deep-translator
- System Integrations: pyperclip

Local Installation & Setup

1. Clone the Repository:
   git clone https://github.com/dharanimahazy/CodeAlpha_Projects.git


   cd "Task 1_Language Translation Tool"

2. Install Required Extensions:
   pip install -r requirements.txt

3. Run Application Runtime:
   python main.py
